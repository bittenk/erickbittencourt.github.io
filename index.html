Landing Page para Barbearia

O desafio era desenvolver uma presença online para uma barbearia, criando uma landing page moderna e atraente. O objetivo era apresentar os serviços, o horário de funcionamento e a localização de forma clara, ajudando a barbearia a atrair novos clientes.

Desenvolvi uma landing page completa, utilizando HTML5 para a estrutura e CSS3 para o design e estilização, com um foco em design responsivo para que o site funcione perfeitamente em qualquer dispositivo. O resultado é uma página intuitiva e visualmente agradável, que serve como uma vitrine digital para o negócio.

Landing Page para Salão de Beleza

O desafio foi desenvolver uma presença online para um salão de beleza, criando uma landing page elegante e visualmente atraente. O objetivo era transmitir a sofisticação e a qualidade dos serviços, apresentando o salão de uma maneira que atraísse um público que valoriza a beleza e o bem-estar.

Eu criei uma landing page completa, utilizando HTML5 para a estrutura e CSS3 para o design. Fui cuidadoso ao selecionar uma paleta de cores, tipografia e imagens que reforçassem a identidade visual do salão. O projeto foi construído com design responsivo, garantindo uma navegação fluida em qualquer dispositivo.
